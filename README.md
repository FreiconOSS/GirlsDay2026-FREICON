## Was machen wir hier?

Drei Schülerinnen entwickeln gemeinsam mit KI-Unterstützung eine Browser-App — in nur einem Tag!  
Die App wird am Ende im Internet zugänglich sein.

## Regeln & Rahmenbedingungen

- **Technik:** HTML + CSS + JavaScript (kein Build-Tool, direkt im Browser)
- **Grafik:** 2D-Fokus — keine aufwendigen 3D-Grafiken
- **Speichern:** Kein Server, kein Backend — alles läuft im Browser (localStorage möglich)
- **Zeitrahmen:** Ein Arbeitstag
